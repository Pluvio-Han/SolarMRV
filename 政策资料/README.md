# 政策资料索引

本目录收集了与 `SolarGuard` 项目申报材料直接相关的官方政策与权威解读，优先覆盖四条主线：

1. `RWA` 与代币化监管边界
2. 数据要素与可信数据空间
3. 数字金融
4. 绿色金融与绿证

当前建议的项目合规表述：

- 不将项目表述为面向中国大陆的 `RWA` 发行或代币交易平台。
- 将项目表述为 `分布式光伏可信数据采集、联盟链存证、绿色资产核验与金融服务辅助平台`。
- 将 `SLG`、`mUSD`、`DEX` 等内容定位为机制设计验证或研究型原型，不作为当前落地业务表述重点。
- 强调项目服务于 `数据确权`、`审计追溯`、`绿证衔接`、`绿色金融辅助决策`、`ESG 披露`。

## 建议在申报书中如何使用

- `政策环境` 章节：优先引用 `01_RWA监管` 和 `02_数据要素与可信数据空间`。
- `项目合规性` 章节：明确写出“本项目现阶段不在中国大陆开展公开代币发行、交易或募资业务”。
- `技术先进性` 章节：引用 `可信数据空间技术架构`、`数据要素x三年行动计划`。
- `市场前景` 章节：引用 `数字金融` 和 `绿证` 相关文件，说明新能源数据、绿色金融、ESG 披露存在真实需求。
- `产品定位` 章节：写成“新能源场景中的可信数据与绿色金融服务平台”，不要写成“代币交易平台”。
