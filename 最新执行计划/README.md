## 三、正式写申请书时的推荐使用顺序

1. 先读 `04_国省级大创提交清单与时间表.md`，确认截止日期和硬性要求。
2. 再读 `计划书写作准备/00_材料总索引与使用顺序.md`，确定写作材料来源。
3. 用 `计划书写作准备/01_旧项目可继承成果清单.md` 提取旧项目硬成果。
4. 用 `计划书写作准备/02_旧项目风险内容与改写规则.md` 清理旧 RWA 高风险表达。
5. 用 `计划书写作准备/03_新旧项目映射表.md` 保证项目升级有延续性，而不是像完全换题。
6. 按 `计划书写作准备/04_申请书章节素材池.md` 起草正式申请书正文。
7. 用 `计划书写作准备/05_附件证据与引用清单.md` 补齐截图、报告、代码、政策和测试证据。

## 四、当前最重要的判断

国省级申报不是简单把旧校级材料润色一下，而是要完成一次**合规叙事切换**：

- 旧项目主线：光伏 RWA、两链一桥、跨境融资、代币 / 积分、交易与收益。
- 新项目主线：分布式光伏 MRV、可信采集、国密签名、联盟链存证、减排量核算、第三方核查接口。

但这不是完全换项目。旧项目里真正有价值的部分是：

- 真实光伏设备采集；
- SM2 / SM3 签名；
- FISCO-BCOS 联盟链存证；
- 自动监测、图表生成、链上同步；
- 8 小时连续测试；
- 前端可视化与云端部署；
- 已形成的政策资料、BP、图表、代码和项目报告。

新申请书要做的是：把这些真实成果从高风险金融化故事里“剥出来”，重新装配成一个合规、可信、可执行的 MRV 数据底座项目。
