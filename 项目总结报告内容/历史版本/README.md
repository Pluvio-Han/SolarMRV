# 历史版本（项目早期阶段材料存档）

本目录归档项目**早期原型阶段**形成的商业计划书、项目报告与方案材料。这些材料以"光伏 RWA / 现实世界资产代币化"为技术探索口径，**不代表项目当前定位与对外口径**。

## 归档背景

2026 年 2 月，中国人民银行、中国证监会等八部门进一步明确了对现实世界资产代币化（RWA）相关业务的监管口径。结合监管环境变化，团队主动完成项目定位的合规规范化调整：

- **早期定位（本目录内文件所对应）**：光伏 RWA / 绿电资产链上化探索
- **当前定位**：分布式光伏 MRV（监测—报告—核查）可信数据底座

调整后的项目**不开展任何金融化资产发行与流转、不撮合交易、不开展公众募资、不承诺收益**，仅作为底层可信数据基础设施服务于 CCER 申报、第三方核查、园区碳管理、ESG 披露与 CBAM 绿电出口证明等场景的数据支撑。
