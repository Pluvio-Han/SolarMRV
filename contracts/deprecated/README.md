# Deprecated Contracts（早期技术原型存档）

本目录归档了项目早期原型阶段开发的三份智能合约。**这些合约已停止部署、停止调用，仅作为早期技术原型阶段的代码留痕，不构成当前项目的功能组成部分。**

## 归档背景

2026 年 2 月，中国人民银行、中国证监会等八部门进一步明确了对现实世界资产代币化（RWA）相关业务的监管口径，强调境内开展代币发行、公开募集、类证券化流转等可能触及非法金融活动监管红线。

为严格遵守政策导向，团队主动完成项目定位的合规规范化调整：

- **从**：早期原型阶段的"光伏环境权益数字化"技术探索口径
- **调整为**：分布式光伏 MRV（监测—报告—核查）可信数据底座

调整后的项目**不开展任何金融化资产发行与流转、不撮合交易、不开展公众募资、不承诺收益**。链上记录仅作为发电数据与减排量核算过程的可信留痕，不可转让、不可流通。

## 归档清单

| 文件 | 原功能（已停用） | 停用原因 |
|------|------------------|----------|
| `SolarRWA.sol` | 早期 RWA 资产管理合约原型 | 监管收紧后停止部署 |
| `SolarToken.sol` | 早期绿电积分代币合约原型 | 监管收紧后停止部署 |
| `StableCoin.sol` | 早期价值结算合约原型 | 监管收紧后停止部署 |

## 当前在用合约

项目当前在用的智能合约仅有一份：

- **`../SolarDataStore.sol`**：纯数据存证合约，提供 `storeDeviceData`、`getDeviceData`、`verifySignature` 等只读 / 写入接口，不含任何价值转移、铸造、兑换逻辑。

后续项目周期内将新增减排量电子记录合约（设计为只读、不可转让、不可销毁），仅作为 MRV 核查链路的数据留痕。
